# Moistello Documentation

Welcome to the Moistello documentation. Here you'll find everything you need to know about using the platform, creating savings circles, and understanding how everything works.

## What is Moistello?

Moistello is a decentralized rotating savings and credit association (ROSCA) platform built on the Stellar blockchain. Members form savings circles, contribute USDC or XLM each cycle, and one member receives the pooled payout per round. Smart contracts enforce rules, handle payouts, apply penalties, and track on-chain reputation.
